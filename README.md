# CatHMI

### 🐾 Project Overview
CatHMI is a specialized Android application I developed in 2022. It serves as the primary bridge—or Human-Machine Interface—between a user and an animatronic feline companion. This project was designed to facilitate seamless interaction, allowing users to control movements, triggers, or responses from the animatronic hardware via a mobile device.

### 🛠️ Technical Details
Built natively for the Android platform, this repository showcases my early work in hardware-software integration. 

* **Platform:** Android (Java)
* **Role:** Human-Machine Interface (HMI)
* **Core Function:** Acting as a controller and feedback system for animatronic hardware.
* **Development Era:** This project was created in 2022, focusing on mobile-to-hardware communication protocols.

### 🚀 Purpose & Vision
The goal of CatHMI was to explore how mobile devices could make robotics more accessible and interactive. By shifting the complexity of robot control to a familiar smartphone interface, it allowed for more intuitive and expressive interactions with animatronic systems.
